## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| **命令转发** | 接收 XServer 命令，转发到 Colo 侧组件 |
| **数据转发** | 收集 Colo 侧数据，转发到 XServer |
| **性能监控** | 监控 CPU、内存、网络、延迟 |
| **进程管理** | 启动/停止/重启 Colo 侧进程 |

## 📊 监控指标

- **网络延迟**: Ping/Latency 监控
- **进程状态**: CPU/内存使用率
- **订单延迟**: Tick2Order 时间
- **系统负载**: Load Average
